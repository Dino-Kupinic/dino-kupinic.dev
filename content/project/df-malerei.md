## Description

### Background

The client is a local painting company that wanted to migrate from their old website to a new one, reasons being yearly costs of the old website hitting close to 2.000€ yearly with poor ranking on search engines and a very outdated design.

### Solution

The solution was to cancel the old contracts and build a new website from scratch. With the goal to reduce yearly costs to a minimum and improve the online presence of the company.

#### New Website

The new website is built with **Next.js** and is hosted on a VPS. The website is designed to be fast, responsive and SEO-friendly.
Compared to the old site it also features:

- A modern design that is more appealing to users
- Better SEO optimization to improve search engine rankings (including Agent Search Optimization (`llms.txt`), `ogImage`, `JSON-LD`, etc.)
- Integration with Analytics tools to track user behavior and website performance
- Improved security and performance with Cloudflare

The website is also optimized for mobile devices, ensuring a seamless experience for users on all platforms.

#### Added Features

Features that were added to the new website include:

- A blog section to share news and updates about the company and the industry
- A career section to attract potential employees and showcase the company culture
- A references section to showcase previous work and build trust with potential customers
- A FAQ and "About us" section to provide more information about the company and its services
