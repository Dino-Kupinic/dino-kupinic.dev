DigiBon is a mobile app prototype for digitizing receipts, developed as part of my diploma thesis for [BMD](https://bmd.com).

### Video demo

:::ProseVideo{src="/images/project/digibon.mp4"}
Video preview is unavailable in this browser.
:::

### Features

- Handling of XML files containing receipt data
- User-friendly interface for viewing and managing digitized receipts
- Mobile-first and responsive design for mobile devices
- PWA features for offline access and installation on mobile devices
- Integration with Supabase for data storage and user authentication
- Customizable receipt categories and tags for better organization
- Search and filter functionality for easy retrieval of digitized receipts
- Charting and analytics features to visualize spending patterns and trends

### Development

It was co-developed with a colleague, where I was solely responsible for the frontend development, while my colleague handled the backend development.
