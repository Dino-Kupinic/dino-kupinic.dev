## Description

Minesweeper is a single-player puzzle game.
The objective of the game is to clear a rectangular board containing hidden "mines" without detonating any of them.

![Preview](/images/project/minesweeper_preview.png)

### Features

- Multiple difficulty levels
- Score tracking
- Timer
- Flagging mines
- 8-bit pixel art style

### Development

This project was a school project, where we had to implement the game logic and a GUI using [JavaFX](https://openjfx.io/).
Styling is done via CSS, and the project is designed in the MVC (Model-View-Controller) architecture.

The assets were made in [Aseprite](https://www.aseprite.org/).
