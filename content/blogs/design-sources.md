There's a saying that goes: "Good artists copy, great artists steal."
The same applies to design where inspiration is everywhere, and the best designers know where to look.

_This is a curated list of some of the best sources for design inspiration, from websites to tools and everything in
between._

### Mobbin

[Mobbin](https://mobbin.com/) is a collection of handpicked apps that reflect the best in design.
It's a great resource for designers and developers looking for inspiration or just wanting to see what's out there.

![mobbin](/images/blog/mobbin_preview.png)

Mobbin features real-world examples from published apps, making it an excellent reference point. Its searchable
interface patterns help users quickly find specific elements. The platform is updated regularly with fresh content and
offers high-quality screenshots for deeper insight.

### Curated Design

[curated.design](https://www.curated.design/) is a collection of the best web design examples and resources.
It's a great place to find inspiration for your next project or just to see what's possible in web design.

![curated.design](/images/blog/curated_preview.png)

The designs are hand-picked to ensure quality and creativity. They emphasize current design trends and allow
users to browse through various categories and filters. Like Mobbin, it is regularly updated to stay relevant.

### Dark Mode Design

[Dark Mode Design](https://www.darkmodedesign.com/) is a showcase of the best dark mode designs from around the web.

![dark](/images/blog/dark_preview.png)

This site focuses specifically on dark themes, offering valuable insights into accessibility. It also provides
inspiration for color palettes and highlights modern design patterns suited for dark UI.

### Footer Design

[footer.design](https://www.footer.design/) is a collection of the best footer designs from around the web.

![footer](/images/blog/footer_preview.png)

Footers. The most underrated part of a website.

### Minimal Gallery

[Minimal Gallery](https://minimal.gallery/) is a collection of the best minimal design examples from around the web.

![minimal](/images/blog/minimal_preview.png)

Probably the best website for minimal design inspiration.
It features a wide range of categories, including startups, portfolios, SaaS and more.
The designs are hand-picked to ensure quality and creativity.
The site is regularly updated with fresh content, making it a great
resource for designers and developers looking for inspiration.

### Saaspo

[Saaspo](https://saaspo.com/) is a collection of the best SaaS design examples from around the web.

![saaspo](/images/blog/saaspo_preview.png)

Saaspo focuses specifically on SaaS design, providing valuable insights into user experience and interface design for software products.
It offers a wide range of examples, from landing pages to dashboards, making it a great resource for SaaS designers and developers.

### Craftwork

[Craftwork.design](https://craftwork.design/) is a collection of the best design resources and inspiration from around the web.

![craftwork](/images/blog/craftwork_preview.png)

Craftwork.design offers a variety of design resources, including UI kits, icons, and illustrations.
It also features a curated collection of design inspiration from various categories, making it a valuable resource for designers and developers alike.

### Component Gallery

[Component Gallery](https://component.gallery/) is a collection of the best UI components and design patterns from around the web.

![component](/images/blog/component_gallery_preview.png)

Component Gallery focuses on UI components and design patterns, providing a wealth of inspiration for designers and developers looking to create intuitive and visually appealing interfaces.

### 21st

[21st.dev](https://21st.dev/) is a collection of UI components and design patterns.

![21st](/images/blog/21_preview.png)

You can copy and paste the code for the components directly into your project, making it a great resource for developers looking to quickly implement design patterns and UI components.

### uiverse

[uiverse](https://uiverse.io/) is a huge collection of a multitude of UI components.

![uiverse](/images/blog/uiverse_preview.png)

Perhaps the largest collection of UI components on the web, uiverse offers a wide variety of design patterns and components that can be easily integrated into projects.
It's a great resource for developers looking for inspiration or ready-to-use components.

## Conclusion

Inspiration is everywhere, and the best designers know where to look.
